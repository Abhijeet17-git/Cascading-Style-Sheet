CSS Overflow Practice

Visible

This content spills out because the box is too small. It overlaps everything below it. This is usually what you DON'T want in a professional layout.

Hidden

This content is "chopped off." You can't see the rest of the text, and you can't scroll to find it. Useful for images!

Auto (The Best)

The box stays neat! Because the text is too long, a scrollbar appears automatically. This is how chat apps and comment sections work.
